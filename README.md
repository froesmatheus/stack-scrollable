# StackScrollable

Scrollable `UIStackView` using `UIScrollView`.

## Usage

```swift
let stack = StackScrollable()
```

### Add view

```swift
stack.add(view: view)
```

### Remove view

```swift
view.removeFromSuperview()
```

## APIs

```swift
func add(view: UIView)
func add(views: [UIView])
func add(views: UIView...)
```
